# Ameya Pratap Singh — Personal Portfolio

A sleek, responsive, and minimalist single-page portfolio website showcasing my projects, skills, and academic background.

🔗 **Live Link:** [personal-portfolio-rho-eosin.vercel.app](https://personal-portfolio-rho-eosin.vercel.app/)

## 🚀 Features
- **Project Showcase**: Visual cards showcasing my main projects (InlineExplainer, SubTrack, Vendora) with live links and source repositories.
- **Interactive Timeline**: Clean tab-switched tabs organizing my About Me section, Education history, and Technical Skills.
- **Responsive Layout**: Designed for optimal viewing across mobile, tablet, and desktop screens.
- **Clean Aesthetics**: Minimalist dark-themed design with custom hover transitions and a sleek scroll-to-top navigation shortcut.
- **Zero Dependencies**: Built entirely with vanilla web standards for performance.

## 🛠️ Built With
- **Frontend**: HTML5, CSS3 (Vanilla Custom Properties), JavaScript (Vanilla ES6)
- **Icons**: Devicons CDN
- **Fonts**: DM Sans & Syne via Google Fonts

## 💻 Local Setup

To run this project locally, simply clone the repository and start a local development server:

```bash
# Clone the repository
git clone https://github.com/AmeyaSingh23/personal-portfolio.git

# Navigate to the folder
cd personal-portfolio

# Start a local Python server (or use VS Code Live Server)
python -m http.server 8000
```
Open your browser and navigate to `http://localhost:8000`.
